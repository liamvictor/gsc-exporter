Google Search Console API Ideas

Account Wide Report

Github: gsc-exporter.

  • Device Performance Breakdown

    • Gather clicks, impressions, CTR, and average position segmented by device type (desktop, mobile, tablet) for each site. This would reveal cross-site device-specific trends.

  • Country Performance Breakdown

    • Include performance metrics broken down by the top X countries for each site. This could highlight geographic strengths and weaknesses across the account.

  • Top Queries/Pages

    • For each site, identify and list the top N queries and/or top N pages based on clicks or impressions. This would offer a snapshot of key content driving traffic across the portfolio.

  • Month-over-Month/Yea

    • over-Year Comparisons: Add columns or sections indicating percentage changes in clicks, impressions, CTR, and position compared to the previous month or the same month in the previous year.

  • Search Appearance (Ric

    • Results): Track and report on the presence and performance of different rich result types (e.g., FAQs, How-to, Product snippets) for each site.

  • Property Type and Verificatio

    • Status: Include metadata for each site, such as whether it's a URL-prefix or Domain property, and its verification status within GSC. This provides useful context.

  • Error/Exclusion Tracking

    • Summarize any significant GSC errors or excluded pages for each property (e.g., number of crawled but not indexed pages, server errors)

  • Average Query Position

    • Distribution: Instead of just average position, show the distribution of queries by position ranges (e.g., top 3, 4-10, 11-20, etc.) for each site.
