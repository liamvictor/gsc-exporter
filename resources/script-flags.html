performance-analysis.py

  • <site_url> (positional argument)
  • --start-date
  • --end-date
  • --last-24-hours
  • --last-7-days
  • --last-28-days
  • --last-month (default)
  • --last-quarter
  • --last-3-months
  • --last-6-months
  • --last-12-months
  • --last-16-months
  • --compare-to-previous-year: Compares the selected date range to the same period in the previous year.
  • Filtering Flags:
  • --page-contains <substring>: Include pages containing the substring.
  • --page-exact <url>: Include only an exact page URL.
  • --page-not-contains <substring>: Exclude pages containing the substring.
  • --query-contains <substring>: Include queries containing the substring.
  • --query-exact <phrase>: Include only an exact query phrase.
  • --query-not-contains <substring>: Exclude queries containing the substring.
