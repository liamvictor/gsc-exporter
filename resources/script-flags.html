Identified Anomalies

Keyword Cannibalisation: Missing --use-cache support. This report can be slow to download, so caching would be a significant improvement.
Query Segmentation: Only supports --last-month and custom dates. Should ideally support the full range of standard date flags.
Global Token Flag: The interactive-runner.py now attempts to pass a --token flag to all scripts, but currently none of the underlying report scripts define this argument in their parser.
