How to Read the Query Position Distribution Report

This report provides a detailed breakdown of your Google Search Console performance by query position ranges, offering a more granular view than just average position. It is generated by the query-position-analysis.py script.

Note: This script can be run for a single site or for all sites in your account. The output format will vary slightly, with the account-wide report including an index of all sites.

Understanding the Position Ranges

  • Positions 1-3 (Top 3): These queries rank at the very top of the search results. High clicks and impressions in this range indicate strong visibility.
  • Positions 4-10: Queries on the first page, but further down. They are valuable but have room for improvement.
  • Positions 11-20: Queries on the second page. They represent opportunities to push onto the first page.
  • Positions 21+: Queries ranking beyond the second page. These highlight long-tail opportunities or areas needing significant SEO effort.

Interpreting the Data

For each site and month, you will see clicks and impressions distributed across these ranges. This allows you to:

  • Identify Strengths: See which sites have a high proportion of traffic in the top position ranges.
  • Spot Opportunities: Look for sites with many impressions in positions 4-10 or 11-20, as these are good candidates for optimization.
  • Monitor Trends: Observe how the distribution changes over time. Are queries moving into higher positions, or are they slipping?
